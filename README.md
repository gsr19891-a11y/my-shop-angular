# My Shop Angular Project

A web-based e-commerce project developed to demonstrate proficiency in Frontend development using Angular.

## 🚀 Technology Stack
- **Framework:** Angular 18+
- **Language:** TypeScript
- **State Management:** Angular Signals
- **API Communication:** HttpClient (RxJS)
- **Styling:** SCSS
- **Automation:** n8n

## 🛠 Features
- Dynamic product catalog with category filtering.
- REST API integration for real-time data retrieval.
- Efficient state management using Angular Signals.
- Responsive UI design optimized for all devices.

## 📦 Getting Started
1. Clone the repository:
   `git clone https://github.com/gsr19891-a11y/my-shop-angular.git`
2. Navigate to the project directory:
   `cd my-shop-angular`
3. Install dependencies:
   `npm install`
4. Start the development server:
   `ng serve`
5. Open your browser at: `http://localhost:4200`
